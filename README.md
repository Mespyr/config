My semi-decent linux configs for my computer.
- Window Manager: DWM
- Terminal: Alacritty
- Text Editor: Emacs

### Install Guide

Build and install `dwm` and `dmenu`.
Copy all the config files.
Run `fc-cache` to update all your fonts.
